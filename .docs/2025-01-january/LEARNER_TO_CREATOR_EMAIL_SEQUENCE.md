# Learner → Creator Conversion Email Sequence

## Overview
A 5-email sequence designed to convert existing learners into creators on PPR Academy.

**Target Audience:** Users with accounts who have:
- Completed at least 1 course OR
- Purchased samples/presets OR
- Been active in the last 90 days

**Goal:** Get learners to create their first product (sample pack, preset, or course)

## Automation Rules

### Trigger Conditions
- User has account for 14+ days
- Has not created a store
- Has either:
  - Completed 1+ courses, OR
  - Made 1+ purchase, OR
  - Logged in 5+ times

### Exit Conditions (Stop Sequence)
- User creates a store → Send "Welcome Creator" email instead
- User unsubscribes
- User hasn't opened any of last 3 emails

### Available Personalization Variables (from Admin Panel)
These are the ONLY variables currently available in your email system:
- `{{firstName}}` - User's first name
- `{{name}}` - User's full name
- `{{email}}` - User's email address
- `{{storeName}}` - Creator's store name (for creator emails only)
- `{{storeSlug}}` - Store URL slug (for creator emails only)

**NOT currently available** (would need to be added):
- ~~`{{coursesCompleted}}`~~ - Doesn't exist
- ~~`{{samplesDownloaded}}`~~ - Doesn't exist
- ~~`{{level}}`~~ - Doesn't exist

### Follow-Up After Sequence
If user completes sequence without converting:
- Wait 30 days
- Send single "We Miss You" re-engagement email
- If still no conversion, add to monthly creator spotlight newsletter only
